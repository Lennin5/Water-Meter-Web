Water Meter

Es un proyecto creado y diseñado desde cero, con el propósito de mejorar el manejo y distribución del agua en los hogares, ocupando las tecnologías para lograrlo. Utilizando nuestros conocimientos de programación y ensamblaje, para combinar software y hardware con agua.


Este proyecto funciona en base a tres partes principales, siendo estas:

Hardware: Contamos con un sensor de flujo de agua, para la medición del agua enviando datos al arduino UNO, para posteriormente mandar dichos datos vía Bluetooth a nuestra aplicación.

Software: Por su parte el software cuenta con una aplicación móvil y una pagina web, las cuales manejan los datos del consumo de agua enviados por el sensor en tiempo real. Primeramente la aplicación, es el controlador principal de todo el sistema "Water Meter", ya que en ella se controlan todos los procesos principales como: llevar un registro del consumo de agua, apoyado de un buscador de datos en base a datos reales de ANDA. Por otra parte, nuestro portal web es un agregado donde se visualiza de manera más sencilla y a la vez detalla, los datos de consumo de agua independientemente de estadísticas del consumo que se realiza en el hogar.

Lógica: Por esta parte ocupamos la lógica matemática para poder desarrollar soluciones para este proyecto, Tales como la tranformación de L a m³ y el manejo de datos de cobro de ANDA por medio de matemática básica y investigaciones de distintas ramas.
